
# 🌍 Wanderlust

A full-stack travel platform to explore, create, and review travel destinations.

## 🚀 Features
- Authentication & Authorization (Passport.js)
- CRUD for Listings & Reviews
- Image Upload (Cloudinary + Multer)
- Interactive Maps (Mapbox)
- Search & Filters
- Responsive UI

## 🛠 Tech Stack
- Node.js, Express.js
- MongoDB (Mongoose)
- EJS, Bootstrap
- Cloudinary, Mapbox

## 📦 Setup
```bash
npm install
npm start

# 🌍 Wanderlust – Full Stack Travel Platform

Wanderlust is a full-stack travel listing and review platform where users can explore, create, review, and manage travel destinations. The platform provides secure authentication, cloud-based image uploads, interactive maps, and a responsive user experience.

# ✨ Features

## 🔐 Authentication & Authorization
- User Signup & Login
- Secure Authentication using Passport.js
- JWT-based Authorization
- Protected Routes & Access Control
- Session Management

## 🏕️ Listings Management
- Create Travel Listings
- Edit & Update Listings
- Delete Listings
- Dynamic Destination Pages
- RESTful CRUD Operations

## ✍️ Reviews & Ratings
- Add & Delete Reviews
- Ratings System
- User-linked Reviews

## 🗺️ Maps & Location Services
- Interactive Maps using Mapbox API
- Geocoding Support
- Dynamic Destination Visualization

## 🖼️ Image Uploads
- Multer for File Uploads
- Cloudinary Cloud Storage
- Image Preview & Management

## ⚠️ Error Handling & Validation
- Custom Express Error Middleware
- Async Error Handling
- Joi Schema Validation
- Client-side Form Validation
- Custom Error Classes

## 📱 Responsive UI
- Bootstrap Responsive Design
- EJS Templating
- Reusable Boilerplate Layouts
- Custom CSS Styling

## ☁️ Deployment
- MongoDB Atlas Integration
- Render Deployment
- GitHub Version Control
